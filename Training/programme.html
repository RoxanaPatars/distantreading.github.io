Building the ELTeC: creating marked up texts

Objectives and prerequisites

In this one day workshop you'll learn:

  • what TEI XML markup is and why we use it for the ELTeC
  • how ELTeC texts should be marked up, at levels 0 and 1
  • some practical ways of creating ELTeC texts starting from other formats,for example :
    • plain text output by an OCR system
    • digital documents created with Word or Open Office
    • ePub or HTML documents found on the internet

You should have some previous experience of working with digital documents, for example creating web pages, or editing documents with Word, but you are not expected to know anything about TEI or XML!

Almost all the practical work will be done using oXygen XML Editor. If you don't already have it installed, you may like to download an evaluation copy of this software before the workshop.
